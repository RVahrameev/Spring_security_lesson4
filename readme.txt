Структура html
index.html
    |_ info.html - страница с информацией о текущей аутентификации
    |_ кнопка "Authentificate by JWT token" - отправляет на сервер запрос на /auth с корректным JWT токеном
    |_ кнопка "Authentificate by bad JWT token" - отправляет на сервер запрос на /auth с отозванным JWT токеном

Единственный тест использован для генерации 2RSA ключей, JWT токена  и записи всего этого файлы в папку ресурсов.
Эти файлы используются при запуске основной программы

Создано два сервисных класса
    RsaKeyManager - подгружает из каталога ресурсов файл с RSA ключами, по которым потом осуществляет проверку JWT токенов
    TokenRevocationCheckService - для ведения списка отозванных токенов и проверки по нему токенов на запрос аутентификации
